# Personal Project 1 : 유튜브 특정 영상 댓글 크롤링 / 모든 댓글 좋아요 누르기 프로젝트

## User : 본인 유튜브 댓글을 주기적으로 분석해야 하는 유튜버 혹은 유튜버의 직원 / 유튜브 댓글에 꾸준히 '좋아요'를 눌러야 하는 유튜버.

## "유튜브 특정 영상 댓글 크롤링 / 모든 댓글 좋아요 누르기 프로젝트" 완료 정리 회고.

- 처음 시작은 취직을 위한 개인 프로젝트 겸 내 실력을 향상 시키기 위한 경험으로써 선택
- Python을 다루는 사람이 업무 자동화를 구현하는건 필수라 생각하여 선택.
- 작은 기능을 구현하는데도 생각보다 많은 작업을 해야했음.

* Selenium을 사용한 동적 웹페이지 크롤링 구현
* 인터렉티브 element 요소를 찾아서 '좋아요' 클릭을 구현하는 방법에서 특히 애먹음.
* Tkinter를 사용함으로써, 사용자에게 보여지는 UI / 프런트는 백엔드 및 비즈니스 논리 로직
  구현과는 다른결의 작업이라는 것을 체감함.
* MultiThreading을 사용해서 작업 속도를 감소시키는 것은 물론, 반드시 멀티스레딩이
  필요한 작업에서 구현을 함.
* 순환 참조, 테스트 프레임워크 내 import Error등 새로운 문제와 해결을 경험함.

- 최초의 'self 요구사항 정의 / 설계 / 구현 / 테스트' 경험이였음.
- 유지보수를 통해 추가해야할 기능이 몇가지 생각이 되며, 짚고 넘어가고 싶지만 시간상
  넘어간 부분들도 꽤 있음. 이 부분도 추후 정리가 필요함.
- 추후 프로젝트에는 실용성 있는, 실제로 user가 있을법하고 유용하고 유효한 프로젝트를
  기획해보는게 중요하다 생각함.
- 이상 Personal Project 1 회고를 마침.
